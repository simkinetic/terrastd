# terrastd
Standard library functionality for the Terra programming languange. The following functionality is supported:
* Allocators, smart unique pointers, weak pointers, etc
* Method dispatch with static interfaces via c++20-style concepts
* Method dispatch with dynamic interfaces
* Lambda's
* Ranges
* Math functions for single, double and high precision floats
* Dynamic stack (similar to C++ std::vector)
